  • 用 flexbox 时 css 基本套路
    父元素居中用下面的写法

  • 子元素加 float,与之相近的父元素加 clearfix 可以进行水平的布局

  • 父元素用相对定位，子元素用绝对定位，那么子元素就相对父元素定位了
  • 负边距一般用于水平垂直居中，需要知道 width/height

  • 如果有多个块级元素，我们想让其排列在一行并且总宽度依然相对父元素居中

  • 行内元素水平居中对其父元素使用 text-align: center
